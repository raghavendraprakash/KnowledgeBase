
# VTU 6th semester, Computer Science syllabus for Machine Learning

**VTU 6th Semester Computer Science Machine Learning syllabus (2022 scheme, applicable 2025-2026) uses course code BCS602 or BCM601.**[^1_1]

It totals 50 hours of pedagogy (4:0:0:0 L:T:P:S), with 50 CIE marks, 50 SEE marks, 100 total marks, and 4 credits.[^1_1]

## Course Objectives

The course introduces machine learning fundamentals, types, challenges, data handling, algorithms like regression and decision trees, Bayesian models, clustering, neural networks, and reinforcement learning.[^1_1]
Students learn to model and evaluate ML solutions for real-world problems using tools like Python.[^1_1]

## Module Breakdown

- **Module 1 (8 Hours)**: ML introduction (need, types, challenges, process, applications); data understanding (Big Data framework, descriptive stats, univariate/bivariate/multivariate analysis).[^1_1]
- **Module 2 (8 Hours)**: Multivariate stats, feature engineering, dimensionality reduction; learning theory basics (system design, concept learning, modeling).[^1_1]
- **Module 3 (8 Hours)**: Similarity-based learning (KNN, weighted KNN, LWR); regression (linear, multiple, polynomial, logistic).[^1_1]
- **Module 4 (8 Hours)**: Decision trees (classification, impurity measures, regression); Bayesian learning (Bayes theorem, classification).[^1_1]
- **Module 5 (8 Hours)**: Clustering (hierarchical, partitional, divisive); reinforcement learning (components, Q-learning).[^1_1]


## Course Outcomes

Students can describe ML techniques and data frameworks; apply feature engineering and reduction; develop similarity/regression models; build probabilistic/neural models; use clustering and reinforcement learning.[^1_1]

## Textbooks

1. S. Sridhar and M. Vijayalakshmi, *Machine Learning*, Oxford University Press, 2021.[^1_1]
2. M.N. Murty and V.S. Ananthanarayana, *Machine Learning: Theory and Practice*, Universities Press, 2024.[^1_1]

Official syllabus PDFs are available on VTU's site under the 2022 scheme for CSE 6th sem.[^1_1]

# VTU 6th semester, Computer Science syllabus for Blockchain

**VTU 6th Semester Computer Science Blockchain syllabus falls under the 2022 scheme as a Professional Elective (BCS613A - Blockchain Technology).**[^2_1][^2_2]

It covers 40 hours of pedagogy (3:0:0:0 L:T:P:S), with 50 CIE marks, 50 SEE marks, 100 total marks, and 3 credits.[^2_2]

## Course Objectives

The course teaches blockchain fundamentals, Bitcoin operations, Ethereum smart contracts/DApps, and Hyperledger platforms for decentralized applications.[^2_1][^2_2]
Students gain skills to design blockchain-based architectures and apply them in real-world scenarios.[^2_2]

## Module Breakdown

- **Module 1**: Distributed systems (CAP theorem, consensus); blockchain intro (history, elements, features, tiers, applications, benefits/limitations).[^2_1][^2_2]
- **Module 2**: Decentralization methods, smart contracts, DAOs; cryptography (symmetric/asymmetric, hashes, Merkle/Patricia trees, ECDSA).[^2_1]
- **Module 3**: Bitcoin (transactions lifecycle/structure, blocks, network, wallets); smart contracts (history, Ricardian, oracles, DAO).[^2_2][^2_1]
- **Module 4**: Ethereum (network, wallets, nodes, smart contracts, DApps development).[^2_2]
- **Module 5**: Hyperledger (Fabric, Sawtooth, Corda) for blockchain apps.[^2_2]


## Course Outcomes

Students can explain blockchain concepts/applications; illustrate principles and smart contract lifecycles; demonstrate Bitcoin methods; develop Ethereum DApps; use Hyperledger architectures.[^2_1][^2_2]

Official details appear in VTU's 6th sem CSE scheme PDF; check vtu.ac.in for the latest 2022 scheme syllabus.[^2_4][^2_2]

# VTU 6th semester, Computer Science syllabus for Cloud Computing

VTU 6th semester Computer Science Cloud Computing syllabus (2022 scheme) uses course code BCS601.[^3_1][^3_2]

It totals 40 hours of pedagogy (3:0:0:0 L:T:P:S), with 50 CIE marks, 50 SEE marks, 100 total marks, and 3 credits.[^3_7][^3_1]

## Course Objectives

The course covers distributed systems foundations, virtualization techniques, cloud architectures, security measures, and emerging cloud applications.[^3_2][^3_1]
Students learn to design, deploy, and secure scalable cloud solutions using platforms like AWS, Azure, and Google App Engine.[^3_4][^3_2]

## Module Breakdown

- **Module 1**: Scalable computing over the internet; technologies for network-based systems; system models for distributed/cloud computing; software environments; performance, security, and energy efficiency.[^3_3][^3_1]
- **Module 2**: Virtualization of CPU/memory/I/O, virtual clusters, data center automation; virtualization structures and mechanisms.[^3_2][^3_4]
- **Module 3**: Cloud computing/service models; data center design and networks; compute/storage cloud architecture; public platforms (GAE, AWS, Azure); inter-cloud resource management.[^3_1][^3_4]
- **Module 4**: Cloud security threats, risks, privacy, data encryption; database/OS/VM security measures.[^3_4][^3_2]
- **Module 5**: Cloud applications, future trends, serverless computing; platforms like AWS Lambda and GCP.[^3_1][^3_2]


## Course Outcomes

Students can explain distributed/cloud models and virtualization; analyze cloud architectures/services; implement security strategies; develop/deploy cloud-native apps.[^3_2][^3_1]

Official syllabus is in VTU's 2022 scheme CSE 6th sem PDF on vtu.ac.in.[^3_7]
